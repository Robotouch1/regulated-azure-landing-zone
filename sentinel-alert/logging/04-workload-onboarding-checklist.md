# Workload Security Onboarding Checklist (Regulated)

## Required before production go-live
- Subscription placed in correct MG (Prod/NonProd)
- RBAC assignments validated (least privilege)
- Policy compliance baseline passed (or exemptions approved)
- Logging enabled (Tier 0 must-have)
- Sentinel onboarding confirmed (incidents visible)
- Defender for Cloud enabled and assessed
- Break-glass + PIM validated for operations
- Evidence artifacts captured (policy compliance snapshot, logging config)

## Outputs
- Onboarding record stored (NDA-safe reference)
- Exceptions recorded if any
